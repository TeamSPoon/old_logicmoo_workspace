This is a demonstration implementation of an OWL macro system. See the examples and Makefile in the above directory.

To run this you will need to either install Thea2, or download Thea2 and place the modules in your library path; e.g.

~/.plrc:
==
user:file_search_path(library, '/users/me/pllib').
==
