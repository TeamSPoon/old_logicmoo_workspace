PURPOSES:

 

 

Manually test operators (to reach the goal state).

 

 

BASIC REQUIREMENTS:

 

 

1. at least one task should be pre created.


 2. operators should be checked by clicking on the menu item "Operators" under the menu of "Validation" (use to be "Semantic Checks").
 
 

TYPICAL PROCEDURE OF USING STEPPER:

 

 

3. select a task from the task combo box on the left, the content (initial and goal states) of the task will appear. The graphical representation of this task is also shown on the canvas.


 4. double click on the operator user wants to be executed from the "Operators List" on the right hand side of the window. An operator Instantiation Window will appear. Click on the down arrow will expend the window to show the detail of this operator.

 5. right click on the varibles, a popup menu will be actived to allow user to select potential object. An soon as an object is selected, the operator is instantiated simultaneously.

 6. click on "OK" button to commit. If all necessary conditions and prevail conditions are met, the resulting states after execution of the current operator will be displayed on the canvas. The changed object state will appear in different colour distinguish between prevail condition, necessary change and conditional change.

 7. repeat step 4 to 6 till a message appears to congratulate user that the goal has been reached.
 
 

GENERAL FUNCTIONS

 

 

8. double click on any object states (small round rectangle), the state property of the current object will be displayed.


 9. double click on any operators (oval shape), the instantiated operator's property will be displayed.

 10. Use "Pre" and "Next" button to go to the previous steps and next steps (if there is one).

 11. All shapes and link lines support the common functions such as click select, drag select, zoom in/out, drag/move, etc.

 12. Any small windows, toolbars can be hidden by selecting the relative mode in "View" functions of the menu bar.

 13. The graphics of the state transitions can be saved as a pure graphical file and can be printed out to a ps file or a printer.
