PURPOSES:

 
Create predicates.

 
BASIC REQUIREMENTS:

 
    1. at least one valid object exists.

 
TYPICAL PROCEDURE OF USING PREDICATE:
     
    1. type string text into the text box "Name", then click "New" button to add the new predicate to the predicate list.
    2. to build members of the predicate, drag sort/object from the sort tree window to the new created predicate in  the "Edit Predicate" window.
    3. when editing an existing predicate, select the predicate from the predicate list, then it's content will appear in the edit window. Repeat previous precedure to edit.
    4. select a predicate, click the check box "Is Static" to make it static/non-static.
    5. to delete a predicate, first select it from the list, then click on the "Delete" button.
    6. to rename a predicate, first select the proper predicate to bring it's content to the edit window. Gently click on the node once, the predicate name node will be changed to the editable mode.
    7. The ordering of sorts in a predicate being edited can be changed by drag and drop to the new positions.
GENERAL FUNCTIONS
     
    1. select the radio button, then click button "Refresh Selection" will narrow the predicate list against the selected object/sort in the sort tree window.
