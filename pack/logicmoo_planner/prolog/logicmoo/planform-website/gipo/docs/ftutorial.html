TUTORIAL
 (flat domains - using basic editors)

Purpose: to create a version of the "briefcase" domain and run it on integrated planners using GIPO
Purpose: to manually test operators and to automatically animate and step planning results using GIPO

Background to the briefcase world

In this world, there are:

2 movable containers: a suitcase and a briefcase.


3 things which can be put into/taken out of any container: a suit, a dictionary, and a cheque.
This is constrained by the fact that the suit only fits into the suitcase.

2 locations: home and office.

rules:
Things can only be moved from one location to another by being carried inside a movable container.

Steps to go

create the domain
perform validity checks
run the stepper
run an integrated planner
view planner results in the animator
