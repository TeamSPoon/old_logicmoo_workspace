Stepper
PURPOSE:

 

 
 
 

Graphically show a planner's result.

 

 
 
 

BASIC REQUIREMENTS:

 

 
 
 

1. a planner has been executed (under menu bar "Plan") and a planning result was produced successfully.

 

 

 2. operators should be checked by clicking on the menu item "Operators" under the menu of "Validation" (use to be "Semantic Checks").
 
 

TYPICAL PROCEDURE OF USING STEPPER:

 

 
 
 

3. when the "Animator window" is opened, the task when the planner was executed is displayed on the left hand side of the window. Operators selected and instantiated by the planner are listed in the right order on the right hand side of the window.

 

 

 4. click on the button "Next" to show the planning result till a message appears to congratulate user that the goal has been reached.
 
 

GENERAL FUNCTIONS

 

 
 
 

5. double click on any object states (small round rectangle), the state property of the current object will be displayed.

 

 

 6. double click on any operators (oval shape or on the "Operators List"), the instantiated operator's property will be displayed.

 7. Use "Pre" and "Next" button to go to the previous steps and next steps (if there is one).

 8. All shapes and link lines support the common functions such as click select, drag select, zoom in/out, drag/move, etc.

 9. Any small windows, toolbars can be hidden by selecting the relative mode in "View" functions of the menu bar.

 10. The graphics of the state transitions can be saved as a pure graphical file and can be printed out to a ps file or a printer.
