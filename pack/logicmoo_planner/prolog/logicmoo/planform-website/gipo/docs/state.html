PURPOSE:

 
Create state.

 

 
 
 
 
 

BASIC REQUIREMENTS:

 
  1. at least one sort/object and one predicate exists.

 
TYPICAL PROCEDURE OF USING STATE:

 

 
 
 

to create a new state
 

  1. from the predicate list, select a predicate, click "Add" button to add it to the "Edit State" window (edit window).
  2. repeat last procedure to add all essential predicates to the edit window.
  3. click on the variables of a predicate in the edit window will highlight this variable with green underline, other associated variables (with same sort) will be also highlighted but with red under line.
  4. right click on the green underlined variable will bring up a menu to rename this variable.
  5. right click on the red underlined variable will allow user to choose from three options: same, different, optional. Choose "same" will set this red underlined variable to be the same as the green underlined variable; Choose "different", an input box will appear to ask for a new name if the red underlined variable is currently the same the green underlined variable, and a "ne(red underlined variable, green underlined variable)" will be added automatically; Choose "optional", an input box will appear to allow user to type in a new name, but no "ne" clause will be added.
  6. to remove a predicate from the edit window, click on the variable of the predicate, then click on the "Remove" button.
  7. when editing finished, click "Add" button to add it to the state list on the right hand side of the window.


to edit an existing state
 

  1. click on the state from the state list ("State Definitions" window), it's content - predicate(s) will appear in the edit window.
  2. repeat the editing procedure described above.
  3. click "Update" button to update the current state.


to delete an existing state
 

  1. click on the state from the state list ("State Definitions" window), it's content - predicate(s) will appear in the edit window.
  2. click on the button "Delete".
